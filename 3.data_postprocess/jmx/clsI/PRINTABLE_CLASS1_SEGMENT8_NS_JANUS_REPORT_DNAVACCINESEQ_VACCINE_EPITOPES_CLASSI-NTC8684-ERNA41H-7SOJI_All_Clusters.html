* Count of IVAX JanusMatrix matches found in the search database. A JanusMatrix match is a 9-mer/10-,er derived from the search database (typically the human genome) which shares TCR facing contacts with an EpiMatrix Hit (a 9-mer/10-mer peptide predicted to bind to at least one HLA) contained in the input sequence and which is predicted to bind to the same allele as an EpiMatrix Hit.

** Janus Homology Score represents the average depth of coverage in the search database for each EpiMatrix hit in the input sequence. With respect to a given EpiMatrix Hit (a 9-mer/10-mer contained within the input sequence which is predicted to bind to a specific allele), a Janus Matrix Hit is a 9-mer/10-mer derived from the search database (typically the human genome) which is predicted to bind to the same allele as the EpiMatrix Hit and shares TCR facing contacts with the EpiMatrix Hit. For example, an input peptide with eight EpiMatrix hits, all of which have one match in the search database, has a Janus Homology Score of 1. An input peptide with four EpiMatrix Hits, all of which have two matches in the search database, has a Janus Homology Score of 2.
